# Implementação de uma aplicacao WEB 

Implementação de um sistema de cadastro de clientes para uma empresa de sucos chamada KiSucos.
Nesse projeto utilizamos as seguintes tecnologias:
 ▪**HTML**
 ▪**CSS**
 ▪**JS**
 ▪**JAVA**
 ▪**SPRING BOOT**
  ▪**MySQL**
